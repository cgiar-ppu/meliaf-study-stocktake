# MELIAF Study Stocktake

A CGIAR tool for cataloguing and managing monitoring, evaluation, learning, and impact assessment (MELIA) studies.

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
